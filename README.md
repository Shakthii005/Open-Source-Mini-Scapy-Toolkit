# Open-Source-Mini-Scapy-Toolkit

A beginner-friendly **network security toolkit** built with [Scapy](https://scapy.net/).  
This project demonstrates **host discovery, port scanning, and packet sniffing** ‚Äî three fundamental skills for any **network security engineer**.
----------------------------------------------------------------------------------------------------------------------------------------
Features
- **Ping Sweep** ‚Üí discover live hosts in a subnet.
- **TCP SYN Scan** ‚Üí check if specific ports are open/closed/filtered.
- **Packet Sniffer** ‚Üí capture packets using BPF filters (e.g., ICMP, TCP, DNS).
